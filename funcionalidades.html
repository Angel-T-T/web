1. Creación de Modales

Un modal es una ventana emergente útil para mostrar mensajes o formularios.

Este es un ejemplo de modal.

<button onclick="showModal()">Abrir Modal</button> <div id="modal" style="display: none;"> <p>Contenido del modal</p> <button onclick="closeModal()">Cerrar</button> </div>

2. Efectos con Parallax

El efecto Parallax crea profundidad moviendo fondos a diferentes velocidades.

background: url('image.jpg') no-repeat center center / cover; animation: pulse 3s infinite;

3. Scroll Animado

Agrega efectos al hacer scroll en la página para captar la atención del usuario.

/* Código CSS para scroll animado */ @keyframes slideIn { ... } animation: slideIn 1s ease;
